## 使用技術詳細說明
- 以HTML、CSS及JavaScript撰寫，為一頁式靜態網頁
- 使用Bootstrap表格（Table）樣式美化表格
- 運用Flexbox操縱部分元素排版
- 使用ES6、jQuery使JavaScript語法變得簡潔（箭頭函式、簡化DOM控制等）
- 採用UJS寫法，維持HTML簡潔
- 運用AJAX fetch外部（政府資料開放平臺）API，再使用ES6方法擷取其中重要資料，展示於本專案頁面
